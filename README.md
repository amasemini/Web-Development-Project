# Web-Development-Project
Project using HTML, CSS, PHP and MySQL (WAMP Server).

# Project Overview

This is a web-based application developed using HTML, CSS, PHP, and MySQL, running on WAMP Server.
The project demonstrates basic web development concepts including front-end design, server-side scripting, and database management.

# Features

* User-friendly interface
* Responsive web pages using HTML & CSS
* Database connectivity using MySQL
* CRUD operations (Create, Read, Update, Delete)
* Form validation
* Dynamic content handling using PHP

# Technologies Used

* HTML
* CSS
* PHP
* MySQL
* WAMP Server (MySQL + PHP)
